# python-lc

Python project with LangChain and JupyterLab (data ingestion).

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirement.txt
```

## Push to GitHub

1. **Create a new repository** on GitHub: [github.com/new](https://github.com/new)  
   - Name it e.g. `python-lc`  
   - Leave "Add a README" **unchecked** (you already have one)

2. **Add remote and push** (run in this project folder):

```bash
git remote add origin https://github.com/aiarpitvarshney/python-lc.git
git push -u origin main
```

Use your repo name if different from `python-lc`.
